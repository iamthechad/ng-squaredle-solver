What is this?

This application is designed to help with solving Squaredle puzzles.

Squaredle is a daily word puzzle that can be played here: https://squaredle.app/

This application is based very heavily on the existing solver at https://github.com/minoli-g/squaredle-solver. All of the actual logic to determine available words for a particular board originated from this project.

Based on the above logic, I created a new version that cleaned up a few minor issues I had with the original version. Thinking this might be a fun project to learn new things with, I'm using Angular with Material for the UI. The project uses NX for the monorepo structure, GitHub Actions for CI build/deploy and is hosted on AWS using CloudFront.

If you're interested in the source, it can be found at https://github.com/iamthechad/ng-squaredle-solver
